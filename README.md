## Previsão de Estoque com SageMaker Canvas

Este repositório contém um projeto que exemplifica a construção de um modelo de previsão de estoque utilizando o **Amazon SageMaker Canvas**, uma plataforma de Machine Learning sem código que permite treinar, avaliar e gerar previsões de forma visual e automatizada.

A partir de um dataset de vendas com variáveis como data, loja, produto, preço e quantidade vendida, o modelo foi treinado por meio de **AutoML**, com análise de métricas de desempenho e identificação das variáveis mais relevantes para a previsão.

O principal objetivo do projeto é apoiar decisões de **planejamento de estoque**, capturando tendências temporais, padrões de consumo e sazonalidade sem a necessidade de programação, facilitando o uso de IA por profissionais de negócio.

O repositório inclui o conjunto de dados utilizado, a documentação completa do processo no Canvas (passo a passo, métricas e saídas geradas) e exemplos de previsões, permitindo a replicação ou adaptação da solução para cenários semelhantes de previsão de demanda e estoque.

Link: https://github.com/rhayannematos/lab-dio-previsao-estoque-sagemaker-canvas

## Projeto ETL de Filmes

Este repositório contém um projeto que implementa um pipeline de **ETL (Extract, Transform, Load)** em Python para o processamento de dados de filmes, desenvolvido como parte do **Desafio ETL do Bootcamp Santander 2025 da DIO**.

Utilizando a biblioteca **pandas**, o pipeline realiza a extração de um arquivo CSV com informações de filmes, aplica transformações como cálculo da idade dos filmes, padronização e organização das colunas, e gera um dataset final limpo e estruturado.

As etapas do projeto refletem práticas essenciais de engenharia de dados, abrangendo a extração dos dados de entrada, a transformação para normalização e enriquecimento das informações, e o carregamento do resultado final para uso analítico.

O repositório inclui o código completo, instruções de execução e o arquivo final de saída, servindo como um exemplo prático e reutilizável de um pipeline de dados em Python.

Link: https://github.com/rhayannematos/etl-transformacao-dados-desafio-dio

## Dashboard de Vendas – Xbox Game Pass

Este repositório contém um **dashboard de vendas desenvolvido no Microsoft Excel** para análise do desempenho do Xbox Game Pass, criado no contexto do **Bootcamp Santander 2025 – Ciência de Dados com Python**.

O dashboard utiliza dados relacionados a assinantes, receita, tipos de planos e outros registros relevantes, incorporando **tabelas dinâmicas, gráficos interativos, segmentadores e KPIs**, como total de assinantes, receita total, ticket médio, crescimento mensal e taxa de renovação.

O objetivo do projeto é transformar dados brutos em **insights acionáveis**, permitindo a identificação de padrões de consumo, tendências no comportamento dos assinantes e suporte visual à tomada de decisões estratégicas.

O repositório inclui o arquivo Excel pronto para uso, com instruções de visualização e detalhamento das abas de dados, facilitando a exploração das métricas e a reutilização para fins analíticos ou de portfólio.

Link: https://github.com/rhayannematos/santander-2025-dashboard-vendas

## Relatório de Implementação de Serviços AWS

Este repositório contém um projeto prático desenvolvido como parte de um desafio da **DIO**, com foco na implementação de serviços da **Amazon Web Services (AWS)** para automação, modernização de infraestrutura e otimização de custos em uma empresa fictícia.

O projeto documenta a utilização de **Amazon S3** para armazenamento escalável com políticas de ciclo de vida, **AWS Lambda** para automação de processos recorrentes em ambiente serverless e **Amazon RDS (MySQL)** para migração de um banco de dados físico para um serviço gerenciado.

Cada serviço foi aplicado a um cenário realista, resultando em maior disponibilidade, redução de esforços operacionais e melhoria da eficiência da infraestrutura.

O repositório inclui o relatório completo do projeto, com objetivos, etapas de implementação, resultados obtidos, diagramas de arquitetura, comparativos de custos e scripts utilizados, permitindo fácil compreensão e replicação da solução.

Link: https://github.com/rhayannematos/dio-aws-project-lab

## VibeFinance – Documentação do Projeto

Este repositório contém a documentação do **VibeFinance**, um aplicativo web de finanças pessoais desenvolvido durante o **Bootcamp CAIXA – Inteligência Artificial na Prática da DIO**, utilizando a abordagem de **Vibe Coding**.

O projeto foi criado para auxiliar usuários no controle financeiro, permitindo o cadastro de receitas e despesas, acompanhamento de saldo, definição de metas de economia e geração de insights inteligentes sobre hábitos financeiros, por meio de uma interface moderna e interativa.

O desenvolvimento explorou a integração da IA como parceira de codificação, acelerando a prototipação, a criação de telas e a implementação da lógica de interação, reforçando boas práticas de usabilidade e design centrado no usuário.

O repositório inclui documentação detalhada, protótipos e exemplos de telas, apresentando o VibeFinance como um caso prático de aplicação de IA no desenvolvimento de software.

## Fake Natty: Conteúdo Natural Criado por IA

Este repositório contém um artigo reflexivo desenvolvido no contexto do **desafio Fake Natty da DIO**, abordando o tema *“Inteligência Artificial: quando a tecnologia começa a parecer humana”*.

O projeto demonstra o uso da **IA generativa como apoio à escrita**, explorando organização de ideias, estruturação do texto, ajustes de linguagem e refinamento do tom narrativo, com foco em produzir um conteúdo fluido e natural.

O artigo propõe uma reflexão crítica sobre autoria, criatividade, ética e os impactos da geração automática de textos no futuro da comunicação.

O repositório apresenta tanto o conteúdo final quanto o processo de construção, evidenciando a IA como ferramenta de apoio criativo e intelectual.

Link: https://github.com/rhayannematos/desafio-fake-natty-artigo-ia-dio
